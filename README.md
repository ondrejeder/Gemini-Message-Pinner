# Gemini Message Pinner

A browser extension that allows users to seamlessly pin and save important interactions across Gemini chat sessions. Pinned prompts are stored persistently in Chrome/Vivaldi sync storage and easily accessible from a sliding sidebar.

## Features
* Pin prompts directly from the Gemini interface.
* Persistent storage across browsing sessions.
* Cross-device syncing.

## Installation
1. Clone the repository.
2. Open your chromium-based browser and navigate to `chrome://extensions/` or `vivaldi://extensions/`.
3. Enable "Developer mode".
4. Click "Load unpacked" and select the extension directory.
